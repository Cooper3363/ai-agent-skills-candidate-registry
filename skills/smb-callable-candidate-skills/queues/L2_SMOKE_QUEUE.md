# 测试台候选 Smoke / L2 队列

## 当前可测范围

本目录当前已有 2 个 `draft_l3` 和 3 个 `component_only`，无需重复正式 L2。后续新增候选先进入 smoke test，再按许可证结果进入正式 L2。

## Smoke Test 目标

- 判断候选是否能用中文 mock 输入稳定输出。
- 判断能否表达成 callable Skill 或组件。
- 判断权限边界和人工复核触发是否清楚。
- 判断是否与 13 个稳交付 Skill 重复。

## 正式 L2 目标

正式 L2 必须满足：

- 至少 3 个中文业务用例。
- 固定输入输出 schema。
- 明确失败判定。
- 明确权限边界。
- 明确人工复核触发。
- 结论只能为：L2 通过可封装、需补测、暂缓、未通过、仅组件。

## 禁止动作

- 不安装依赖。
- 不访问真实账号。
- 不调用真实 API。
- 不发送邮件、短信或微信。
- 不写 CRM、日历、任务或业务系统。
- 不抓取网页。
- 不生成真实图片、视频、音频、OCR 或 PDF 结果。
